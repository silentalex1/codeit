How to use your API key

Use the standard OpenAI-compatible endpoint:

POST https://codeit-7u2k.onrender.com/v1/chat/completions

Headers:
Authorization: Bearer YOUR_API_KEY
Content-Type: application/json

Body:
{
  "model": "prysmisai",
  "messages": [{"role": "user", "content": "Hello!"}],
  "stream": true
}

This endpoint uses the embedded model running inside the app (same as the website chat).
