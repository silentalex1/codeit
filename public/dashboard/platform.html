Your PrysmisAI API Keys

These keys let any app call Gemini 2.5 Pro through PrysmisAI without exposing your Gemini key.

/v1/chat/completions (OpenAI compatible)

curl https://YOUR-SITE/v1/chat/completions \
  -H "Authorization: Bearer PrysmisAI_xxxxxxxxxxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gemini-2.5-pro",
    "messages": [{"role":"user","content":"Write a Roblox round system in Luau"}],
    "stream": true
  }'

/v1/messages (Anthropic compatible)

curl https://YOUR-SITE/v1/messages \
  -H "Authorization: Bearer PrysmisAI_xxxxxxxxxxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gemini-2.5-pro",
    "messages": [{"role":"user","content":"Build a clean datastore wrapper"}],
    "max_tokens": 4096
  }'

Using from code (Python example)

from openai import OpenAI
client = OpenAI(base_url="https://YOUR-SITE/v1", api_key="PrysmisAI_xxx")
r = client.chat.completions.create(model="gemini-2.5-pro", messages=[{"role":"user","content":"..."}])
print(r.choices[0].message.content)

Important: You must first visit the chat, log in with the same username, and save your personal Gemini 2.5 Pro API key in the modal. Only then will PrysmisAI_ keys work.
